## Using Braintree with Nango[](#using-braintree-with-nango "Direct link to Using Braintree with Nango")

Provider template name in Nango: `braintree`  
If you want to connect to the Sandbox system use `braintree-sandbox`Follow our
[quickstart](/quickstart) to add an OAuth integration with Braintree in 5
minutes.

**Metadata captured by Nango**  
The `merchandId` is automatically stored in the
[Connection Metadata](/reference/core-concepts#metadata) during the OAuth flow.
